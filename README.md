# Dokdistsentralprint
Dokdistsentralprint er en del av Doksys og distribuerer dokumenter til sentral print hos Skatteetaten.

Mer informasjon om hvordan appen fungerer finner du på [Confluence-siden for dokdistsentralprint](https://confluence.adeo.no/display/BOA/dokdistsentralprint).

### Henvendelser
Spørsmål om koden eller prosjektet kan rettes til [Slack-kanalen for \#Team Dokumentløsninger](https://nav-it.slack.com/archives/C6W9E5GPJ).
